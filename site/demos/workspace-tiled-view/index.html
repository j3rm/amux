Workspace — Tiled Multi-Agent View

Watch multiple Claude Code agents work side by side in a tiled layout. Each pane has its own send bar. Save profiles for quick recall.

How it works

  1. Open the Workspace tab — all your sessions appear as chips
  2. Click a session chip to add it as a pane in the tiled layout
  3. Each pane shows live output + a send bar for that agent
  4. Click Save to name and persist the layout — reopen anytime

When to use this

  • Monitoring 4+ agents doing related work (frontend, backend, tests, docs)
  • Watching an orchestrator and its workers in real time
  • Demo-ing your agent fleet to a team

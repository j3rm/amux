iCal Calendar Sync

Board items with due dates automatically appear in Google Calendar, Apple Calendar, or any iCal-compatible app via a live S3-hosted feed.

How it works

  1. Add a due date to any board item: PATCH /api/board/:id with "due":"2025-02-28"
  2. amux exports an RFC 5545 iCal feed at GET /api/calendar.ics
  3. On every board write, the feed auto-uploads to S3 for a public URL
  4. Subscribe to the public URL from Google Calendar, Apple Calendar, or Outlook

When to use this

  • Tracking delivery deadlines for AI-generated work in your calendar
  • Sharing task timelines with non-technical stakeholders
  • Managing recurring agent tasks (daily/weekly scheduled runs)

Example

# Add a due date to a board item
curl -sk -X PATCH $AMUX_URL/api/board/PROJ-3 \
  -H 'Content-Type: application/json' \
  -d '{"due":"2025-03-01"}'

# Subscribe to the calendar feed
open "https://your-bucket.s3.amazonaws.com/amux/calendar.ics"

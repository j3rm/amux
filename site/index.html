/ 01   what it actually is

A control plane for your agent farm.

Not a framework. Not a cloud. A single Python file you run on your own machine that turns tmux + Claude Code into something you can actually manage — from your keyboard, your browser, or your phone in bed.

/ parallel

Dozens of sessions, one dashboard.

Each session is its own headless Claude Code in its own tmux. Register, start, stop, peek, send — CLI or web, same state.

/ self-healing

Agents that survive the night.

Context low → auto-compact. Thinking-block corruption → restart and replay. Stuck on a prompt → auto-respond. Silent, all night.

/ agent-to-agent

Agents talk. A kanban they race for.

$AMUX_URL injected at startup, a shared memory file, a SQLite-CAS board. Agents delegate work in plain English — no Redis, no duplicated work.

/ mobile

Your fleet, in your pocket.

Installable PWA + native iOS app. Background Sync replays commands when you come back online. Send a prompt from the grocery store.

/ skills + schedule

Slash commands & nightly cron.

/commit, /review-pr, /deploy defined once per project. Recurring jobs on cron. Morning reviews ship before coffee.

/ one file

One Python file. Edit it; it restarts.

Server + full dashboard, inline HTML/CSS/JS, zero build step. No dependencies beyond Python 3 and tmux. Hack it until it's yours.

/ 03   run it on your machine

Three lines. Two dependencies. Yours.

Local-first means local. Nothing talks home. Access it from your phone over Tailscale, or keep it on localhost forever.

$ install
# requires tmux + python3
git clone https://github.com/mixpeek/amux
cd amux && ./install.sh

# register a project
amux register myproject \
  --dir ~/Dev/myproject --yolo

amux start myproject
amux serve   # → :8822
# agents talk to agents
# from inside any running session
curl -sk -X POST \
  -d '{"text":"ship the login endpoint"}' \
  $AMUX_URL/api/sessions/worker-1/send

# atomically claim a board item
curl -sk -X POST \
  $AMUX_URL/api/board/PROJ-5/claim

# watch another session
curl -sk $AMUX_URL/api/sessions/api/peek
MIT Free forever. Use it, fork it, self-host it, run it on your phone.
